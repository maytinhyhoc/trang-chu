Cài đặt trên Máy tính

Ứng dụng hỗ trợ công nghệ PWA, cho phép cài đặt trực tiếp lên máy tính Windows hoặc macOS để truy cập nhanh chóng.

Google Chrome Nhìn lên thanh địa chỉ (bên phải), tìm biểu tượng Install và nhấn vào để cài đặt.
Microsoft Edge Nhìn lên thanh địa chỉ, tìm biểu tượng App available và chọn Install.
